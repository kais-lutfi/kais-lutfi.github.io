    • Classes & IDs

    When to use Classes

    Classes are usually applied to multiple elements. If for instance you needed to apply the same "font-family" (ie. the same font style) to particular selected elements, you would use a class.

    How to apply Classes

    In order to apply classes to an element; you need indicate in your CSS sheet your class and the style rules. You could do that by using a period "." and then specifying your attribute name. Unlike HTML you follow with curly bracket "{" to open and "}" to close.

    In order for your HTML file to know where to apply those rules, you need to indicate your "class" within the opening tags. Image below shows how your HTML tags would look like.

    When to use IDs

    IDs are very similar in action to classes, but they differ in that IDs should only be used in one element. For instance, using IDs for headers and footers is more appropriate since they only appear in a page once.

    How to apply IDs

    Similarly to classes, IDs need to have its rules on the CSS sheet. However, IDs are open with number sign "#". For your HTML to apply your ID, "id=idname" need to be applied within your opening tags.
